

## المشكلة

عند رفع ~2000 ملف MSBT، المحرر يعرض فقط "إجمالي النصوص" (~70,000 نص) بدون توضيح عدد الملفات الأصلية. كل ملف MSBT يحتوي على عدة نصوص (entries)، لذا الفرق طبيعي تماماً — لكن المحرر لا يوضح ذلك للمستخدم.

## الحل

### 1. إضافة عداد الملفات في شريط الإحصائيات (Editor.tsx)

في منطقة الإحصائيات العلوية (سطر ~376)، إضافة بطاقة جديدة تعرض عدد ملفات MSBT الفريدة:

```
📄 2,309 ملف MSBT
📝 70,000 إجمالي النصوص  
✅ 48,435 مترجم
```

يتم حساب عدد الملفات الفريدة من `state.entries` باستخدام:
```typescript
const uniqueFileCount = new Set(state.entries.map(e => e.msbtFile)).size;
```

### 2. إضافة عداد ترجمة لكل ملف في فلتر الملفات

في قائمة فلتر الملفات الموجودة، إضافة عدد النصوص المترجمة بجانب كل ملف:
```
filename.msbt (15/23 مترجم)
```

### 3. تحسين صفحة المعالجة (MsbtProcess.tsx)

في صفحة "done"، عرض ملخص واضح:
- عدد الملفات المرفوعة
- عدد النصوص المستخرجة منها
- عدد النصوص المكتشفة كمترجمة

### التغييرات التقنية

| الملف | التعديل |
|---|---|
| `src/pages/Editor.tsx` | إضافة بطاقة "عدد الملفات" في شريط الإحصائيات + تحسين فلتر الملفات بعدد الترجمات |
| `src/hooks/useEditorState.ts` أو `useEditorQuality.ts` | حساب `uniqueFileCount` و `perFileTranslationCount` كـ useMemo |
| `src/pages/MsbtProcess.tsx` | تحسين ملخص صفحة "done" |
